# Auralis SkyShare

Piattaforma per voli privati empty-leg / sky sharing.

## Struttura del progetto

```
auralis-skyshare/
├── index.html       ← frontend (statico, nessun build step)
├── backend/         ← API Node.js + TypeScript (in arrivo, FASE 2)
├── .gitignore
├── .env.example
└── README.md
```

## Stato del progetto

- [x] Frontend esistente (HTML/CSS/JS statico)
- [ ] Backend Node.js + TypeScript
- [ ] Database PostgreSQL (Neon)
- [ ] Autenticazione (Auth0)
- [ ] Pagamenti (Stripe Checkout)
- [ ] Email automatiche (Resend)
- [ ] Deploy frontend (Vercel)
- [ ] Deploy backend (Render)
